EasyRoads3D V3 Manual
 

 

User Guide

This user guide explains all the EasyRoads3D features. To get a first impression it is recommended to go through either the Getting Started section or the Quick Start section. Going through the full manual afterwards and getting familiar with all the features and options will help getting the most out of it. The demo project can be used as a reference, it includes examples of all the features with direct links to the Tutorials section on our website.

Please review the troubleshooting and known issues sections in the case you are experiencing unexpected behaviour.

A general knowledge of Unity is recommended. All generated objects are ordinary Unity GameObjects with a Mesh Filter, Mesh Renderer and Mesh Collider attached.
